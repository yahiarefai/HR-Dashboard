HR Dashboard

Project Overview:

This project presents an interactive HR analytics dashboard built using Microsoft Power BI.  
The dashboard analyzes employee data to provide insights into workforce distribution, training costs, and key HR metrics that support data-driven decision making.

Repository Files :

- HR Dashboard.pbix → Power BI dashboard file  
- README.md → Project documentation  

Key Features & Insights :
The dashboard provides analysis on:
- Employee distribution by department  
- Workforce breakdown by gender and age group  
- Training costs and employee participation  
- Overall HR KPIs for management reporting  

Tools & Technologies :
- Microsoft Power BI Desktop  
- Data modeling  
- Data visualization  
- DAX (for calculated measures, if applicable)  

How to Use :
1. Download the file HR Dashboard.pbix 
2. Open it using Power BI Desktop 
3. Explore the report pages and use the filters to interact with the data  

Project Purpose :
This project was developed as part of my studies in **Business Intelligence** to practice analyzing HR data and designing professional dashboards that transform raw data into actionable insights.
